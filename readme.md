# Backend Installation command

note first establish `.env` file setup

```
PORT=8000
MONGODB_URI=mongodb+srv://username:password@mern-practice.tafoa5w.mongodb.net
CORS_ORIGIN=*
JWT_SECRET=YOURSECRETKEY
JWT_EXPIRES_IN=1d
REFRESH_SECRET=YOURSECRETKEY
```

```
cd backend
npm i
npm run dev
```
